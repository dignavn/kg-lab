## Analysis Question 1

**Question:** What is an advantage of using JSON-LD format? What is an advantage of using Turtle format? For both formats, provide a situation where it is best to use this format over the other.

**Answer:**

JSON-LD is built for web developers and easy to understand for them. It integrates easily into REST APIs or Javascript apps without requiring new tooling or libraries. While building (web) application, JSON-LD is better to use.
The Turtle format however, is more human friendly. It is easier to read when you are not a technical person. Also for writing and debugging, Turtle is easier to use. Especially with a complex RDF graph, Turtle beats JSON-LD. When people will read or edit the data in for instance an triple store or file, Turtle wins.
